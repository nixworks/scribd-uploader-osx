Moving Scribd Uploader to the Applications folder

The first time you run Scribd Uploader, it may ask you if you want to move the program to the Applications folder. This will happen if you run it from a different folder, like the Downloads folder, where it may have been placed after you downloaded it.

Scribd Uploader will work in any folder. You can leave it in the folder where it currently is, put it on the Desktop, or anywhere you please. The Applications folder is the default place for programs on your computer, however, and many people like to consolidate their programs into one folder.

Click the Move button if you would like Scribd Uploader to place itself in the Applications folder along with your other programs. Click this button if you’ve just downloaded Scribd Uploader and haven’t yet moved it yourself. Scribd Uploader will be copied to the Applications folder, the original copy will be placed in the Trash, and the program will be relaunched from the Applications folder.

Click the Don’t Move button if you’ve already placed Scribd Uploader into a folder and you’d like to keep it there. You will not be asked if you want to move Scribd Uploader the next time you launch it.
