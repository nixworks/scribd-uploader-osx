The file was not uploaded because it is empty

This is an upload error that occurs when you give an empty file to Scribd Uploader. An empty file is one with no data in it whatsoever. This is different from, for instance, a Microsoft Word document with no content; that at least has some data telling the computer that it’s a Word document. An empty file has no data, not even enough to be an empty document.

If you didn’t expect the file to be empty, it’s possible that it has become corrupt. If you have a backup of the file, use that instead.
