File types supported by Scribd.com

Scribd.com supports a limited number of file types. This is because your documents are converted into a format that can be viewed by anybody using only a Web browser, so Scribd.com needs to know how to convert from each different file type into this Web-friendly format.

Below is a list of the different file types that Scribd.com supports.

Program or category
Supported file types

Microsoft Office

Word documents (.doc, .docx)

PowerPoint presentations (.ppt, .pptx)

Excel spreadsheets (.xls, .xlsx)

PowerPoint shows (.pps)

Universal formats

Text files (.txt)

PDF documents (.pdf)

PostScript documents (.ps)

Rich text files (.rtf)

OpenOffice

OpenDocument texts (.odt)

OpenDocument presentations (.odp)

OpenDocument spreadsheets (.ods)

OpenDocument graphics (.odg)

OpenDocument formulae (.odf)

StarOffice

StarWriter documents (.sxw)

StarImpress presentations (.sxi)

StarCalc spreadsheets (.sxc)

StarDraw images (.sxd)
