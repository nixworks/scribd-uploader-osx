The file was not uploaded because Scribd.com couldn’t access it

This is an upload error that occurs when Scribd.com can’t access the URL you gave it. It can happen because you mistyped the URL, or because the server that hosts the file is having problems.

Try typing the URL into your web browser to make sure the URL is accessible. If you mistyped the URL, dismiss the error window, then follow the steps below to re-enter it correctly:

Fixing a mistyped URL:

  1. Select the mistyped URL from the file list and click the Remove button.

  2. Select the File > Add URL… menu item.

  3. Retype the URL correctly.

  4. Click the Add URL button.

  5. Click the Upload button to re-upload the URL.
