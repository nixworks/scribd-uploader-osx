The username has already been taken

This is an error that occurs when making a new Scribd.com account if you choose a username that has already been taken by someone else. Each Scribd.com account must have a unique username, so if someone got your username before you could, you will have to choose a different one.

If, for instance, you wanted the username “DallasCowboys,” but someone beat you to it, you might try something like “DallasCowboys1.” Very popular usernames may require you to try several variations before finding one that isn’t taken.

Usernames are not case-sensitive. In other words, Scribd.com sees no difference between “username” and “USERNAME.” If either is taken, you will not be able to use both.

Choose a different username, type it in, and try signing up again.
