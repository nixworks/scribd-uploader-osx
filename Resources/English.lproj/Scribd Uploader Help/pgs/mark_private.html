Making documents private after uploading them

If you don’t want others to see a document, you can upload it privately (see Uploading files that only I can see). If you accidentally upload a document as public, you can still change it to private later using the Scribd.com website. Follow the steps below to learn how.

To change a public document to private:

  1. Log in to the Scribd.com website.

    Visit www.scribd.com in your browser and click Log In in the upper right-hand corner of the web page. For more information, read How to view files you’ve uploaded.

    Skip this step if you see a Log Out link instead of a Log In link.

  2. Click the My Docs link. (It’s the second link on the toolbar that runs along the top of the web page, below the Scribd logo.)

    You will see a list of the documents you uploaded.

  3. Click the check mark to the left of each document that you wish to change to private.

  4. Click the More Actions drop-down and choose Make Private.

  5. Click the Apply button.

The page will refresh and the documents you checked should say “Private” to the right of them.

If your My Docs list spans multiple pages, you will have to repeat this for each page of the list. Check just above the list on the right-hand side. If you see a list of page numbers there, your My Docs list spans multiple pages. Repeat the above steps for each page of the list that contains public documents that you want private.
