Adding a file by URL

Along with uploading files from your hard drive, Scribd Uploader can upload files stored on the Internet or accessible of your local area network. You will then have a copy of this file in your Scribd.com account.

To upload a file by URL to Scribd.com:

Uploading a file by URL

  1. Choose Add URL… from the File menu.

  2. Type the URL of the document you want to upload.

  3. Click Add URL.

    The Add URL window will close and the file will appear in your file list.

  4. Repeat steps 1 through 3 for each additional URL you want to upload.

  5. Click Upload to begin uploading the files to your Scribd.com account.

Your files will then be added to your Scribd.com account. If any file could not be accessed, or if some other error occurs, a will appear next to the file. You can click on it for more information about the error.

Scribd.com will handle downloading and re-uploading the remote file. The file will not be downloaded to your computer.
