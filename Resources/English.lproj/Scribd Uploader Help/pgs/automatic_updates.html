Automatic Updates preference pane

The Automatic Updates pane of the Preferences window allows you to control how and when Scribd Uploader checks the Internet for updates. Staying up-to-date ensures you get bug fixes and new improvements in a timely manner.

  • Automatically check for updates:  
    If you check this box, Scribd Uploader will periodically check for updates in the background as you work. If an update is available, you will be asked if you want to install it.
  • Check for updates:  
    You can choose the frequency of the update check (daily, weekly, or monthly). Checking more frequently ensures you get new updates sooner.
  • Include anonymous system information:  
    Normally, Scribd records no information about you or your computer when the update check is made. If you check this box, Scribd Uploader will send information about your computer and operating system. No personal information or identifying information will be sent. Scribd uses this information to determine what kinds of computers run Scribd Uploader.
  • Automatically download updates:  
    Normally, Scribd Uploader will not download an update until you elect to install it. If you check this box, Scribd Uploader will download the update in the background. That way, if you later elect to install the update, it will be ready for you sooner.

You can manually check for updates by clicking the Check now button or choosing Check for updates from the Scribd Uploader menu.
