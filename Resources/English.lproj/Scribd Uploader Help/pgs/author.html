About the author

The Mac OS X Scribd Uploader is an open-source project led by the developers of the Scribd.com website, employees of Scribd, Inc. The project is primarily maintained by Scribd.com developers, but because it is open source, anyone can contribute to it.

Please contact Scribd for any concerns or questions you have about either the program or the website. Visit Scribd.com’s contact page for information on how to get in touch.
