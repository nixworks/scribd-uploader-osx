Solving upload problems

There are several reasons why your file does not upload. If you are given a specific error message, it may contain clues as to why your file could not be uploaded. Your error might be listed on the Solving problems page. If not, try these steps below.

Try these steps to get your files to upload:

  • Make sure you’re connected to the Internet by checking another application, such as Safari. If your Internet connection is not working, run Network Diagnostics.

    Open Network Diagnostics

  • Try going to www.scribd.com in your web browser.

    Open www.scribd.com in my web browser

    If the web page doesn’t load, Scribd may be temporarily unavailable. Try again in a moment.

  • Try saving the file in a different format. If it’s a Microsoft Word file, for instance, try exporting it to a PDF file, or an older Microsoft Word format.

  • Try uploading your files using the Scribd.com website. You may get more detailed error messages from the website than you will from Scribd Uploader.
