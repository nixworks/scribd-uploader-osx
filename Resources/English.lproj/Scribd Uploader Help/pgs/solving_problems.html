Solving Problems

Troubleshooting tips and other ways to solve issues with this software.

Problems when uploading files

Scribd Uploader does not remove files from the list until they have been successfully uploaded. If one of your documents failed to upload, it will remain in the list for a later time. To view the reason why the document failed to upload, read How to view what went wrong when a file failed to upload.

In general, you might find a solution for your upload problem by reading Solving upload problems. Below are some of the problems that can occur when trying to upload a file.

The file was not uploaded because it no longer exists
The file was not uploaded because you don’t have permission to look at that file
The file was not uploaded because it is empty
The file could not be uploaded because Scribd doesn’t accept files of that type
Problems logging in

Below are some of the problems that can occur when trying to log in to Scribd.com using the application.

Your login and password didn’t match. Please double-check them and try again
Problems creating an account

Below are some of the problems that can occur when trying to create a new Scribd.com account using the application.

You must provide an email address and password to sign up for a Scribd account
The email address is invalid. Please provide a valid, real email address
There is already a Scribd user with that email address
The username has already been taken. Please choose another one
