Adding files to share

There are a couple of convenient ways you can tell Scribd Uploader which files you want to upload. They are listed below with instructions on how to use each method.

Adding files with the Add button:

  1. Click the Add button on the toolbar.

  2. Use the file browser that appears to locate your files.

    If you want to add more than one file in a folder, you can command-click to select multiple files, or shift-click to select a range of files.

  3. Click the Add button in the file browser, or press Return.

    The files should appear in your file list.

Adding files using drag and drop:

  1. In the Finder, open a new browser window and navigate to the location where your files are stored.

    You can skip this step if your files are on the desktop or are otherwise already visible.

  2. Select the files you want to upload.

    If you want to add more than one file in a folder, you can command-click to select multiple files, or shift-click to select a range of files.

  3. Drag the selected files into the large white file list in the center of Scribd Uploader’s main window.

    The files should appear in your file list.

Now that your files are ready to be uploaded, you can click the Upload button to upload them to Scribd.com.

If you have a lot of files to add in multiple folders, see Adding all files in a folder to learn how to quickly add a lot of files.
