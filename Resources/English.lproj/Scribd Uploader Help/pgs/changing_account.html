Switching between multiple Scribd accounts

Normally, Scribd Uploader remembers the last account you logged in with, so that it can skip the log in process for future uploads. If you have more than one Scribd.com account, however, you can use the steps below to switch between them.

To switch to a different Scribd.com account:

  1. Click the Switch User button.

    This button will be titled Log In if you haven’t logged in yet.

  2. Enter the username and password for the account you want to use.

    You can also elect to create a new Scribd.com account here if you wish.

  3. Click the Log In button.

    You will log in as the user you specified and your files will begin uploading.

The information for this new account will be remembered for next time. If you want to switch users again, repeat the steps above.
