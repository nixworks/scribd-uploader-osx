The file was not uploaded because Scribd doesn’t accept files of that type

This is an upload error that occurs when you give Scribd Uploader a file of a type Scribd does not accept. Scribd only accept files of a certain type, generally documents produced by major office productivity software.

Normally, Scribd Uploader will not allow you to add a file type not supported by Scribd.com, but occasionally these files can sneak through anyway. If they do, this error will be shown.

Read What types of files can I upload to Scribd.com? for a list of supported file types.
