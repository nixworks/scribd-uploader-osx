Minimum system requirements

Scribd Uploader requires, at a minimum, the following conditions to be met for you to comfortably use this program:

  • Operating system:  
    You will need to be running Mac OS X version 10.6 (“Snow Leopard”) or later.
  • Internet connection:  
    You will need an Internet connection. A broadband connection is recommended if you are uploading large files.

Scribd optionally uses Growl for notifications. You don’t need to have it installed, but if you want to use Growl, you must be running at least version 0.7.
