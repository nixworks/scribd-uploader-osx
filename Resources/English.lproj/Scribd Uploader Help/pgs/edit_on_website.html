Editing information for the files you’ve uploaded

Normally, once you finish uploading some documents, you are shown a sheet that gives you the option of proceeding to Scribd.com so you can edit your files. If for some reason you would like to edit your files later on, you can follow these steps to do so on Scribd.com.

To edit your newly uploaded files using Scribd.com:

  1. Log in to the Scribd.com website.

    Visit www.scribd.com in your browser and click Log In in the upper right-hand corner of the web page. For more information, read How to view files you’ve uploaded.

    Skip this step if you see a Log Out link instead of a Log In link.

  2. Click the My Docs link. (It’s the second link on the toolbar that runs along the top of the web page, below the Scribd logo.)

    You will see a list of the documents you uploaded.

  3. Click the check box to the left of each document you want to edit.

  4. Click the Edit button.

You will be taken to a page where you can add tags, categories, and other metadata to each document, as well as edit its description and title. When you are finished editing all your documents, click any Save button to confirm your changes.
