The file was not uploaded because you don’t have permission to look at that file

This is an upload error that occurs if Scribd Uploader can’t open a file because it doesn’t have the correct permissions. On Mac OS X, each file has permissions that tell the computer who can see it and who can’t. Most of the time, Scribd Uploader should be able to open and see any file on your computer, but occasionally a file’s permissions can be changed, either purposely or accidentally.

If you suspect that the file’s permissions were changed accidentally, one thing you can try is repairing your disk permissions.

To repair disk permissions:

  1. Open Disk Utility in Applications > Utilities, or by clicking below.

    Open Disk Utility

  2. Read Repairing disk permissions in the Disk Utility help book.
