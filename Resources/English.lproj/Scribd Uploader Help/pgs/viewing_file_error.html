How to view what went wrong when a file failed to upload

If a problem occurs when uploading a file in your list, a icon will appear next to the upload progress indicator. The file will remain in your list and the upload will be retried the next time you click the Upload button.

To learn why an upload failed:

  1. Click the icon next to the file that failed to upload.

    A sheet will appear describing the error.

  2. On the Solving problems page, locate the error description and follow the link to learn more about the error.

  3. Click OK to close the error sheet.

Although unlikely, it’s possible an error that is not covered by this help book will occur. If that happens, you can read the About the author page for information on how to get in touch with Scribd.
